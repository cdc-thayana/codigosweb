# 🚀 PRATICANDO COM PYTHON E JAVASCRIPT

## Versão 3.8

Plataforma web interativa para prática de programação diretamente no navegador.

O projeto permite escrever, executar e testar códigos de:

* 🐍 Python
* 🟨 JavaScript
* 🌐 HTML
* 🎨 CSS

com editor profissional, console integrado e visualização em tempo real.

---

# 📌 Sobre o Projeto

O **Praticando com Python e JavaScript** foi desenvolvido para auxiliar estudantes e iniciantes no aprendizado de programação.

A plataforma funciona diretamente no navegador e possui:

✅ Editor de código Monaco
✅ Execução JavaScript
✅ Executor Python no navegador
✅ Visualização HTML/CSS
✅ Console de resultados
✅ Banco de exercícios
✅ Sistema de salvamento local
✅ Download dos códigos
✅ Estatísticas de progresso

# ▶️ Como Executar

1. Baixe ou clone o projeto:

```
git clone SEU_REPOSITORIO
```

2. Abra o arquivo:

```
index.html
```

3. Execute no navegador.

Recomendado utilizar:

* Google Chrome
* Microsoft Edge
* Firefox
